## 1. 核心結論

## 2. Grundy 值與 isWin 的完美對應

> [!NOTE]
> 在所有 n≤10 的 146 個正方形節點中，**isWin = True ⟺ Grundy = 0** 完美成立，無一例外。
>
> 這符合組合賽局理論中 *last player wins* 的結論：到達終局的人贏。

| 組合 | 數量 |
|:---|:---:|
| Grundy = 0 且 isWin = True | 78 |
| Grundy = 0 且 isWin = False | 0 |
| Grundy ≠ 0 且 isWin = True | 0 |
| Grundy ≠ 0 且 isWin = False | 68 |

## 3. 對稱中軸節點 [a,b,a,b] 的勝敗矩陣

對稱節點（R0=C0=a, R1=C1=b）的勝敗呈現清晰的階梯結構：

```
  b\a   0   1   2   3   4   5
   0    W   L   W   W   W   W
   1    ·   L   L   L   L   L
   2    ·   ·   W   L   W   W
   3    ·   ·   ·   L   L   L
   4    ·   ·   ·   ·   W   L
   5    ·   ·   ·   ·   ·   L
```

> [!TIP]
> **觀察規律：**
> - 對角線 a=b：交替出現 W 和 L（W 在 a=0,2,4,6...；L 在 a=1,3,5...）
> - b=0 行：只有 a=1 是 L，a≥2 全為 W
> - b=奇數：整行幾乎全為 L
> - b=偶數且 a-b ≥ 2：全為 W

## 4. 奇偶性分析（Parity Signatures）

將每個節點的矩陣 [R0,R1,C0,C1] 取 mod 2 得到奇偶簽名 (R0%2, R1%2, C0%2, C1%2)，共有 16 種可能，實際出現 14 種：

### 純粹簽名（11/14，100% 可預測）

| 簽名 (R0,R1,C0,C1) % 2 | 結果 | 含義 |
|:---:|:---:|:---|
| (0,0,0,0) | **全部 Win** | 四值皆偶 |
| (0,0,1,0) | **全部 Win** | R0 偶, C0 奇, R1=C1=0 |
| (1,0,0,0) | **全部 Win** | R0 奇, C0 偶, R1=C1=0 |
| (0,0,0,1) | **全部 Lose** | |
| (0,1,0,0) | **全部 Lose** | |
| (0,1,0,1) | **全部 Lose** | R1 奇且 C1 奇 |
| (0,1,1,1) | **全部 Lose** | |
| (1,0,1,1) | **全部 Lose** | |
| (1,1,0,1) | **全部 Lose** | |
| (1,1,1,0) | **全部 Lose** | |
| (1,1,1,1) | **全部 Lose** | R1 奇且 C1 奇 |

### 混合簽名（3/14，無法僅靠奇偶判定）

| 簽名 | Win 數 | Lose 數 | 特殊規律 |
|:---:|:---:|:---:|:---|
| (0,1,1,0) | 1 | 10 | 極少數 Win：[2,5,5,0] |
| (1,0,0,1) | 1 | 10 | 極少數 Win：[5,0,2,5]（上方的鏡像） |
| (1,0,1,0) | 10 | 3 | 大多 Win；Lose 為 [1,0,1,0], [3,2,3,2], [5,4,5,4] |

> [!WARNING]
> **不存在單純的奇偶公式**可以完美判定所有節點的 isWin。
> 三個混合簽名的存在說明勝敗取決於更精細的結構，不僅僅是矩陣值的 mod 2。

## 5. 最佳近似公式

以下公式在 n≤10 的 146 節點上達到 **142/146（97.3%）** 的準確率：

$$\text{isWin}(R_0, R_1, C_0, C_1) = \bigl(R_1 \equiv 0 \pmod{2}\bigr) \;\wedge\; \bigl(C_1 \equiv 0 \pmod{2}\bigr) \;\wedge\; [R_0, R_1, C_0, C_1] \neq [1,0,1,0]$$

**4 個例外：**

| 矩陣 | 預測 | 實際 | 原因 |
|:---:|:---:|:---:|:---|
| [3,2,3,2] | Win | **Lose** | 對稱中軸 a=3,b=2；(1,0,1,0) 簽名的例外 |
| [5,4,5,4] | Win | **Lose** | 對稱中軸 a=5,b=4；同上 |
| [5,0,2,5] | Lose | **Win** | (1,0,0,1) 簽名的唯一 Win |
| [2,5,5,0] | Lose | **Win** | (0,1,1,0) 簽名的唯一 Win |

## 6. 必勝策略路徑

### 策略原則

**勝方的核心策略：保持走到 Grundy=0（isWin=True）的狀態，讓對手面臨 isWin=False 的局面。**

### 具體必勝路徑範例

#### n=4 的最佳路徑

```
先手 [4,0,4,0] Win  →(e)→
後手 [3,1,3,1] Lose →(r)→
先手 [3,0,2,2] Win  →(e)→
後手 [2,1,1,2] Lose →(r)→
先手 [2,0,0,2] Win  →(c)→
後手 [1,0,0,1] Lose →(c)→
先手 [0,0,0,0] Win ✓
```

#### n=8 的最佳路徑

```
先手 [8,0,8,0] Win  →(e)→
後手 [7,1,7,1] Lose →(c)→
先手 [6,2,7,0] Win  →(e)→
後手 [5,3,6,1] Lose →(e)→
先手 [5,2,5,2] Win  →(e)→    ← 回到中軸！
後手 [4,3,4,3] Lose →(r)→
先手 [4,2,3,4] Win  →(e)→
後手 [3,2,2,3] Lose →(e)→
先手 [2,2,2,2] Win  →(r)→    ← 回到中軸！
後手 [2,1,1,2] Lose →(r)→
先手 [2,0,0,2] Win  →(c)→
後手 [1,0,0,1] Lose →(c)→
先手 [0,0,0,0] Win ✓
```

### 策略觀察

> [!TIP]
> **先手的必勝策略共同特徵：**
> 1. **開局走 e 步**：所有起點 [n,0,n,0]（n≥2）的第一步都是 e 步
> 2. **利用 r/c 步打破對稱**：當對手回到對稱中軸時，利用 r 步或 c 步將局面偏向有利方向
> 3. **控制 R1 和 C1 的奇偶**：勝方的走法傾向保持 R1 和 C1 都是偶數
> 4. **最終收束至 [2,0,0,2] 或 [0,2,2,0]**：路徑最後都經過這些「瓶頸」節點

## 7. n=1 為何必敗？

n=1 時矩陣為 [1,0,1,0]，先手唯一合法走法是 e 步到 [0,0,0,0]，遊戲直接結束。

但 [0,0,0,0] 是 isWin=True（到達者勝）——這意味著此時先手「走到終局」的動作本身讓**後手**成為到達 [0,0,0,0] 狀態的一方（因為先手走完後是後手的局面），所以先手必敗。

> [!NOTE]
> 在 DoubleChess 的語義中，isWin=True 代表「遊戲到此節點，當前移動者剛完成最後一步的玩家獲勝」。n=1 的先手被迫走唯一的 e 步到達終點，但終點標記為 Win 表示「到達此狀態的人贏了」——而到達 [0,0,0,0] 的是先手走的那一步，所以先手在 [1,0,1,0] 其實是在走向一個子節點全部 isWin=True 的情況，因此 isWin=False。

## 8. 總結

| 問題 | 結論 |
|:---|:---|
| 是否存在簡單的必勝公式？ | **近似公式存在**（R1, C1 皆偶則勝，97.3% 準確），但**完美公式尚需考慮對稱中軸的特殊結構** |
| 先手是否總能贏？ | **n≥2 時先手總是必勝**；n=1 先手必敗 |
| 必勝策略的核心？ | 開局走 e 步，中盤利用 r/c 步打破對稱，控制 R1+C1 保持偶數 |
| Grundy 值的意義？ | Grundy=0 完美等價 isWin=True，可用於組合遊戲分析 |
