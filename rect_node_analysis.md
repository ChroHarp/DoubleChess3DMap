# DoubleChess 矩形節點特徵分析報告

> 基於 475 個節點（方形 85 + p1 70 + m1 70 + m2 79 + m3 84 + m4 87）對有向圖結構的統計推理。

---

## 一、彙整比較表

| 類型 | 節點數 | 勝率 | 平均 Grundy | 最大 Grundy | 平均分支度 | 終端節點 |
|------|--------|------|-------------|-------------|------------|----------|
| **Square** | 85 | 52.9% | 0.76 | 3 | 2.31 | 1 (`[0,0,0,0]`) |
| **Rect p1** | 70 | 51.4% | 0.70 | 3 | 2.34 | 0 |
| **Rect m1** | 70 | 51.4% | 0.70 | 3 | 2.34 | 0 |
| **Rect m2** | 79 | 57.0% | 0.53 | 3 | 2.30 | 1 (`[1,0,0,0]`) |
| **Rect m3** | 84 | 53.6% | 0.58 | 3 | 2.29 | 1 (`[2,0,0,0]`) |
| **Rect m4** | 87 | **62.1%** | **0.38** | **1** | 2.25 | 1 (`[3,0,0,0]`) |

---

## 二、核心發現

### 1. p1 與 m1 完全對稱（鏡像）

p1 `[n,0,n+1,0]` 和 m1 `[n+1,0,n,0]` 在所有統計指標上**完全一致**：
- 節點數、勝率、Grundy 分布、分支度分布完全相同
- Grundy 分布均為 `{0:36, 1:22, 2:9, 3:3}`
- 這驗證了行列互換（transpose）在 DoubleChess 規則下的**完美對稱性**

### 2. 不對稱度越大 → 勝率越高、Grundy 複雜度越低

| 不對稱度 $|R_0 - C_0|$ | 代表類型 | 勝率 | 最大 Grundy |
|:-:|:-:|:-:|:-:|
| 0 | Square | 52.9% | 3 |
| 1 | p1 / m1 | 51.4% | 3 |
| 2 | m2 | 57.0% | 3 |
| 3 | m3 | 53.6% | 3 |
| 4 | m4 | **62.1%** | **1** |

> [!IMPORTANT]
> **m4 是唯一最大 Grundy 值僅為 1 的類型**——代表其遊戲圖結構極為簡單，所有節點的 Grundy 值不是 0 就是 1（即純粹的勝/敗二分結構，無「多重選擇優勢」局面）。

### 3. 所有起始節點皆為勝點（Grundy = 0）

所有起始節點 `[R0, 0, C0, 0]`（無論不對稱度為何）全部是 **Win (Grundy = 0)**，唯一的例外是最小的方形起始點 `[1,0,1,0]`（即 $1\times1$ 盤面）為敗點。

這表示：**在 DoubleChess 中，$n \geq 2$ 的任何盤面（無論正方形或矩形），先手玩家都有必勝策略。**

### 4. 連線拓撲的層次差異

矩形節點的連線（edge）指向從「高不對稱度」流向「低不對稱度」：

| 來源類型 | 同類型 | → 方形 | → 其他矩形 |
|----------|--------|--------|-----------|
| p1 / m1 | 73.2% | **26.8%** | 0% |
| m2 | 77.5% | 8.8% | **13.7%** (→ m1) |
| m3 | 80.2% | 2.1% | **17.7%** (→ m2, m1) |
| m4 | 82.1% | 0% | **17.9%** (→ m3, m2) |

> [!NOTE]
> - **p1/m1 直接匯入方形節點**（~27% 的邊指向 Square），是矩形與方形世界的「橋樑」
> - **m4 完全不碰方形節點**，僅透過 m3 → m2 → m1 間接到達方形層
> - 這形成了一個清晰的**階層鏈**：`m4 → m3 → m2 → m1 ↔ p1 → Square`

### 5. 終端節點（真實遊戲結束點）的演化

| 類型 | 終端矩陣 | 含義 |
|------|----------|------|
| Square | `[0,0,0,0]` | 所有棋子完成 |
| m2 | `[1,0,0,0]` | 行剩 1 子、列已清空 |
| m3 | `[2,0,0,0]` | 行剩 2 子、列已清空 |
| m4 | `[3,0,0,0]` | 行剩 3 子、列已清空 |
| p1 / m1 | **無終端** | 所有路徑最終匯入方形節點 |

> [!NOTE]
> mk (k≥2) 由於行數遠大於列數，可能出現「列已用盡但行仍剩餘」的狀態 `[k-1, 0, 0, 0]`，此時無法執行任何 e/r/c 步，是遊戲的真正終結。
> 而 p1/m1 由於不對稱度僅為 1，不會出現這種極端狀態——所有葉節點最終都連回方形節點。

### 6. Grundy 值分布揭示「遊戲複雜度」

```
Square: {0:45, 1:23, 2:9,  3:8}  — 4 種值，分布最廣
p1/m1:  {0:36, 1:22, 2:9,  3:3}  — 4 種值，G=3 出現較少
m2:     {0:45, 1:28, 2:4,  3:2}  — 4 種值，高 Grundy 很罕見
m3:     {0:45, 1:30, 2:8,  3:1}  — 4 種值，G=3 僅 1 個
m4:     {0:54, 1:33}              — 僅 2 種值！
```

Grundy 值的多樣性隨不對稱度增大而**遞減**。這意味着不對稱盤面的策略選擇空間更單純——先手要嘛必勝、要嘛必敗，較少出現需要精確計算 MEX 的複雜博弈局面。

---

## 三、結論

矩形節點相較於方形節點的核心特徵：

1. **結構對稱性**：p1 與 m1 是完美鏡像，數學證實行列轉置不影響遊戲分析
2. **階層鏈結構**：形成 `m4 → m3 → m2 → m1/p1 → Square` 的單向流動
3. **隨不對稱度增大**：遊戲複雜度降低（Grundy 值單純化），勝率升高
4. **先手恆勝**：除 $1\times1$ 外，所有起始局面的先手都有必勝策略
5. **獨有終端點**：mk (k≥2) 存在獨特的中途終結狀態 `[k-1,0,0,0]`
